Getting the most of these Tutorials

As you go through the different lessons and chapters, you should use these tips to maximize your learning.

  • Write your own programs using some more concepts you have learned. Nothing is better the Practice
  • Experiment with the examples. Change numbers and text to see what happens. Make the program shown in that lesson do more things. Try to break the program (Taking it to its limits).
  • As you make mistakes, fix them. Try and solve the problems by yourself and then ask google or someone else. Learn about the Debugger (Future Chapter * Not Scheduled for Release any time soon *).
  • Type out the Program. Don't Copy and Paste it.

Note: The majority of the examples in the tutorials are full programs that you can compile and run yourself. However, occasionally the examples will be "snippets" of code that are designed to quickly illustrate a concept. Because these aren't full programs, they won't compile without some additional work. You can turn these into full programs yourself, if you desire.
